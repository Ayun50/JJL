# 雅思听力高频词汇听写练习
一个专门为雅思学生设计的听力单词听写练习工具，帮助减少因拼写错误导致的失分。无需注册，完全免费，开箱即用。

## 🎯 设计初衷
作为一名雅思老师，我常在给学生复盘模考时发现，很多人在听力部分因拼写错误3-5个单词，至少被扣了0.5到1分。现在很多练习平台这类功能都开始收费了，于是我设计了这款App来帮助我的学生练习听写。今天也把它分享给所有有需要的考生和老师。

## ✨ 主要功能
双词库独立选择 – 两个下拉列表可分别从不同词库（如剑桥雅思真题集、主题分类词汇）中选题，互不干扰

智能拼写框

字母下方自动显示下划线，提示单词长度

支持多词词组（如“traffic jam”），长词组自动换行显示

空格键可以直接跳转到下一个单词输入

发音辅助 – 每次加载单词自动朗读，点击喇叭图标可重复播放

实时反馈

正确/错误即时提示，正确后显示中文释义

“显示答案”按钮可直接揭示正确答案（不统计准确率）

学习统计

记录已练单词数、正确数、正确率（仅首次作答计入）

单词总数显示，计时器从第一次键入开始计时

多设备适配 – 完美支持手机、平板、笔记本电脑到27英寸大屏幕

词库自由扩展 – 只需编辑 words.json 即可添加任意主题和单词

## 🚀 在线体验
将本项目部署到任何静态服务器（如 GitHub Pages、Netlify、Vercel）即可使用。

## ⚙️ 如何使用
从左侧或右侧下拉列表中选择一个词库主题

根据字母框提示输入单词

按回车键或点击 Check 检查答案

正确后按钮变为 下一个，按回车继续

中途可点击 Answer 显示正确答案

统计信息（已练、正确、正确率、计时）实时更新

## 📝 自定义词库
编辑 words.json 文件，格式如下：

json
{
  "list1": {
    "剑桥雅思20": [ ... ],
    "剑桥雅思19": [ ... ]
  },
  "list2": {
    "政府": [ ... ],
    "教育": [ ... ]
  }
}
每个单词包含：

word: 英文单词（支持带空格的词组）

translation: 中文释义（建议带词性，如“n. 政府”）

tip: 提示信息（简短说明）
